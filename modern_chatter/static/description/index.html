FAQ

Common questions

Everything you need to know before installing.

Does it work on Odoo.sh? +

Yes. Fully compatible with Odoo.sh hosting. Install it like any other module from the App Store.

Does it require any configuration after install? +

No. Install and it works immediately on all records — no settings to configure, no fields to add.

Does it work on all models? +

Yes. Works on any model with a chatter — CRM, Sales, Invoices, Projects, Helpdesk, and more.

What's the difference between free and Pro? +

The free version gives you the modern UI, search, stars, emoji reactions, and note editing. Pro adds tab navigation, author & date filters, pinned messages, starred filter, sort toggle, and human-only filter.

Does it change how I send messages or manage activities? +

No. Modern Chatter only replaces the message list view. The compose bar, Log Note, Send Message, Activities, and Followers widgets are completely untouched.
